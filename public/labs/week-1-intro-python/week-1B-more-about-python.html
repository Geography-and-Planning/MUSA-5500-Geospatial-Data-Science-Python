Python Functions

A function in Python is a block of code that only runs when it is called. You can pass data, known as parameters, into a function. A function can return data as a result or not return anything just conduct some tasks.

Python functions structure

    .

    .

    return result

Defining a Function

To define a function, use the def keyword followed by the function name and parentheses. Here’s an example:

def compute_square(x):
    
    sq = x * x
    return sq

sq = compute_square(10)
print(sq)
100

Keywords: arguments with a default!

def compute_product(x, y=5):
    return x * y

# use the default value for y
print(compute_product(3))

# specify a y value other than the default
print(compute_product(5, 10))

# can also explicitly tell Python which arguments are which
print(compute_product(5, y=2))
print(compute_product(y=2, x=5))

Python Modules

Python modules are files containing Python code (functions, classes, variables) that you can reuse in other programs. Modules help organize code and make it reusable.

What is a Module?

A module is simply a .py file. You can import its contents into your script or notebook.

Use the import statement to bring in a module:

import math
import numpy as np
import pandas as pd
import random
import os, os.path
from matplotlib import pyplot as plt

Types of Python Modules

There are two types of Python modules:

  • Built-in Modules: These are modules that come pre-installed with Python, such as math, random, and time.
  • External Modules: These are modules that are installed separately, such as numpy, pandas, and matplotlib.

Python for I/O

Read and Write Text File

In Python, you can easily read from and write to text files using built-in functions. Below is a simple tutorial on how to do this.

Writing to a Text File

To write to a text file, you can use the open() function with the mode set to 'w' (write mode). If the file does not exist, it will be created. If it does exist, it will be overwritten.

# Define the file path
# Get the current working directory
import os

current_directory = os.getcwd()
file_path = os.path.join(current_directory, 'data.txt')

# Write to the file
with open(file_path, 'w') as file:
    file.write("Hello, World!\n") #`\n` means new line
    file.write("This is a text file.\n")
    file.write("Python makes it easy to read and write files.")

Reading from a Text File

To read from a text file, you can use the open() function with the mode set to 'r' (read mode).

# Read from the file
file_path = 'data.txt'
with open(file_path, 'r') as file:
    content = file.read()
    print(content)

Writing to a CSV File

To write data to a CSV file, we can use the to_csv() method provided by the pandas DataFrame. Here’s an example:

# Define the DataFrame to be written to a CSV file
data = {
    'Name': ['Alice', 'Bob', 'Charlie'],
    'Age': [25, 30, 35],
    'Gender': ['F', 'M', 'M']
}
df_to_write = pd.DataFrame(data)
# Define the file path where the CSV will be saved
output_file_path = 'output.csv'
# Write the DataFrame to a CSV file
df_to_write.to_csv(output_file_path, index=False)

Read and write image file

In this tutorial, we will learn how to read and write PNG images using the PIL (Pillow) library and convert them to and from NumPy arrays. You may need to install the module, pip install pillow.

Read image

You can use Python to read an image file of local image file or a URL image file. The image reading is through the module of PIL, we will working on reading geotiff in following labs.

from PIL import Image
import numpy as np

response = requests.get(url)

Writing a PNG Image

To write a NumPy array back to a PNG image, we first convert the NumPy array back to a PIL image using Image.fromarray(), and then save it using the save() method.

# Load the image
url = "https://raw.githubusercontent.com/opencv/opencv/master/samples/data/lena.jpg"
response = requests.get(url)
img = Image.open(BytesIO(response.content))

# Convert the image to a NumPy array, we add a random value to each pixel
img_array = np.array(img) + np.random.rand(*img_array.shape)
# Set all values > 100 to 0
img_array[img_array > 100] = 0
# Convert to uint8
img_array_uint8 = img_array.astype(np.uint8)
# Convert the NumPy array back to a PIL image
img_from_array = Image.fromarray(img_array_uint8)

# # Save it
img_from_array.save('output_image.png')
